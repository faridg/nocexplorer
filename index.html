Classification Structure

Class 0: Management occupations Class 1: Business, finance and administration occupations Class 2: Natural and applied sciences and related occupations Class 3: Health occupations Class 4: Occupations in education, law and social, community and government services Class 5: Occupations in art, culture, recreation and sport Class 6: Sales and service occupations Class 7: Trades, transport and equipment operators and related occupations Class 8: Natural resources, agriculture and related production occupations Class 9: Occupations in manufacturing and utilities
About this project

OntarioJobs.org takes XML RSS feed from Jobbank.gc.ca and brings you the latest job listings in Ontario using PHP cURL library. UI is entirely built on Web Experience Toolkit, an open source project available here.

Just navigate to the Class of interest and choose a job title. Four digit number to the left of the job title is the code that you need for immigration. It is also called NOC.
